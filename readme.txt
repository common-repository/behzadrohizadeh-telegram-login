== Description ==

A Plugin For Login And Register in Wordpress Site  with telegram messanger Account. 


== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Menue > Telegram Login And Save Setting.
4.Use [lg-telegram]  Shortcode For Show Button . 
5.Use `<?php echo do_shortcode('[lg-telegram]') ?>` php code in themeplate.
